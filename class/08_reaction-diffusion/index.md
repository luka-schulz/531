#### Promises
Makes asynchronous programming easy

```js

  if( Math.random() > .5) {
    reject();
  }
  else {
    resolve()
  }
}).then( result => {
  console.log( "hooray")
}).catch( error => {
  console.log( "bad" )
})
```
